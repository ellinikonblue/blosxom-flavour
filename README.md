# blosxom 向け Flavour

　[Ellinikonblue.com](http://www.ellinikonblue.com)/[Ellinikonblue.net](http://www.ellinikonblue.net) で使用している [blosxom](http://www.blosxom.com) 用のフレーバーです。


## xhtml/
　XHTML 1.0 Transitional に準じたフレーバーです。

## html5/
　HTML5 に準じたフレーバーです。

## rss10/
　"rss10" プラグイン用のフレーバーです。
